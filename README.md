# 🎓 University Course Registration System

A Python-based Course Registration System that allows students to register for university courses while enforcing academic rules such as prerequisite validation and credit limits.

This project simulates a real-world university registration workflow using:

Object-Oriented Programming (OOP)\
CSV-based data storage

It is designed as a beginner-to-intermediate level project to demonstrate core software engineering and data handling skills.

⚙️ Features\
Load course data from a CSV file\
Load student academic history\
Register for courses\
Prevent registration if:\
Prerequisite is not completed\
Total credits exceed 15\
View registered courses and total credits\
Clean modular OOP design

🧱 Project Structure

course-registration-system

🧠 System Design\
🔹 Classes Used\
**Course**\
Stores course details (code, title, credits, prerequisite)\
**Student**\
Tracks completed courses\
Tracks registered courses\
Calculates total credits\
**RegistrationSystem**\
Loads course data\
Handles validation logic\
Manages course registration\

Object-Oriented Programming (OOP)\
